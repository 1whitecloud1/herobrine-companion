# Herobrine Companion

他不是一个玩家。他不是一只幽灵。他是你最强大的盟友。

Herobrine Companion 是一款为 Minecraft 1.20.1forge/1.21.1neoforge打造的模组，彻底重塑了传说中的 Herobrine 形象。他拥有复杂的情感、丰富的互动和完整的背景故事，能成为你冒险路上真正"活着"的伙伴。他会观察你的行为，记住你的对话，根据你的选择改变态度，并在你赢得信任后提供超越想象的守护。

## 🌟 核心特性

### 🧠 活着的传说

**动态心态**：Herobrine 会根据你的行为在多种心态间切换——观察者、守护者、审判者、恶作剧者……每种心态都影响他的言语和行为。

**细腻动画**：他会冥想、检视镰刀、调试"代码"、甚至闪烁瞬移，每个动作都充满生命感。

**环境感知**：他能"看见"你在看什么方块、手里拿着什么物品，并做出相应反应。

### 🤝 信任与陪伴系统

**信任等级**：通过交易、共同战斗、完成委托逐步提升信任，解锁新对话和奖励。

**陪伴模式**（信任50级后开启）：Herobrine 会跟随你，根据地形自动切换浮空/步行形态，距离过远时瞬间传送回你身边。herobrine会有特殊语言

**清除障碍**（信任50级）：Herobrine会清除附近地上的杂草与树木，保留原始地形

**抹平地形**（信任70级）：Herobrine会整平附近的地形，保留地形方块特色

### 💬 智能对话系统

**云端自然语言**（可选）：接入 OpenAI 兼容 API，实现真正自由的对话，他会记住你们之前的聊天内容。

**本地对话库**：离线时通过关键词匹配回复，内置数千条预设对话。

**情境触发**：他会根据你的活动（挖矿、战斗、探索）主动点评，或在你生命垂危时发出警告/嘲讽。

### 🌫️ 世界异象与探索

**不稳定地带**：主世界随机生成扭曲区域，出现地狱岩、灵魂土等异常方块，幽灵生物在此出没。

**终末之环**：专属维度，由三层基岩圆环构成，是 Herobrine 的领域。首次进入会触发成就，右键herobrine触发特殊对话并获得一本边界手记。

**幽灵军团**：幽灵僵尸、幽灵苦力怕等虚无生物，只在不稳定地带生成，击杀掉落特殊物品可以与herobrine交易。

**幽灵史蒂夫**：罕见生物，0.5%概率生成在主世界地表，击杀掉落"源代码碎片"。

**故障村民 "N."**：以64绿宝石出售屏障方块，退出交易页面后触发惊悚音效并留下传说碎片后消失。

### ⚔️ 神性武装与遗物

**终末之诗**：Herobrine 的战镰，兼具斧子、锹、镐、锄功能。攻击触发虚空裂缝（范围伤害），右键发射破境雷霆（爆炸）。

**灵魂绑定契约**：死亡不掉落物品。

**凌越之允**：生存模式获得创造飞行能力。

**幽邃之视**：永久夜视。

**回溯之石**：传送至上一次死亡位置。

**记忆碎片**：放入唱片机召唤故障回响，引导寻找稀有结构。

**创世神的契约**：召唤 Herobrine 的核心物品，签署契约后右键召唤。

### 📜 传说收集系统

**传说手册**：记录已解锁的背景故事。

**知识碎片**：共11种，散落世界各地，与手册合并可解锁 Herobrine 的过往，同时他会"学习"这些知识，改变对你的态度。

### 🕊️ 和平誓约与背叛

**和平模式**：前往终末之环后开启，怪物将无视玩家。攻击任何生物会破誓，引来狂暴怪物群。

**君王光环**：Herobrine 在场时，周围怪物会停止攻击，末影龙会降落静默。

战利品分布与获取指南 (Loot Distribution & Acquisition)
为了让玩家能够更有针对性地探索世界并解锁 Herobrine 的完整记忆，我们公开了核心道具的详细生成逻辑与掉落概率。

🗝️ 永恒之钥 (Eternal Key) —— 维度通行证
永恒之钥是前往“终末之环”维度的唯一凭证。根据最新的世界生成规则，它不再是完全随机的掉落物，而是作为顶级宝藏出现在世界遗迹的深处。

生成机制：系统采用了 herobrine_companion:add_item 修改器，确保在特定容器中具有 100% (1.0) 的生成判定权重。

获取地点：

远古城市 (Ancient City)：深暗之域的箱子中。

末地城宝藏 (End City Treasure)：末地船或高塔顶部的箱子中。

要塞图书馆 (Stronghold Library)：存放着古老知识的图书馆箱子中。

生存提示：由于生成概率极高，玩家无需搜遍每一个遗迹，通常在探索完第一个完整的上述结构后即可获得该钥匙。

📜 知识碎片分布图谱 (Lore Fragments Map)
为了拼凑出 Herobrine 的完整起源，玩家需要收集散落在世界各地的知识碎片。不同的碎片对应着世界观中不同层面的“真相”。

🧩 碎片 2：被埋藏的历史 (Fragment #2)
这块碎片记录了被掩埋在地底的秘密。

要塞图书馆 (Stronghold Library)：50% 概率。这是获取该碎片的最佳地点，象征着旧时代的知识残留。

废弃矿井 (Abandoned Mineshaft)：30% 概率。在矿车运输箱中偶尔能发现。

🧩 碎片 3：生物的异变 (Fragment #3)
这块碎片揭示了 Herobrine 的存在对普通生物造成的影响。它通过击杀原版怪物获取。

掉落源：僵尸、骷髅、苦力怕、蜘蛛、女巫、末影人。

掉落率：5% (0.05)。你需要进行大量的战斗来获取这部分数据。

碎片4:进入end ring 维度成功离开回到主世界自动获取到物品栏

🧩 碎片 5：幽灵代码 (Fragment #5)

掉落率：5% (0.05)。由于幽灵生物仅在“不稳定地带”生成，这块碎片的实际获取难度较高。

## 🚀 如何开始

### 遭遇 Herobrine

他会在主世界随机生成，尤其喜欢亮度≤7的黑暗环境，常在玩家附近8-16格出现。

聊天框输入指令 `hb`可立即召唤。

### 交互方式

- 右键他打开交互界面（交易、任务、查看信任等）。
- 手持特定物品右键他，可邀请他"查看位置"、"休息"、"守卫"等（取决于心态）。
- 直接聊天（需配置云端）或通过界面文字交流。

### 提升信任

- 交易稀有材料。
- 完成他发布的任务。
- 解锁传说手册并阅读。
