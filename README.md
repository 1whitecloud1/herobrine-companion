# Herobrine Companion

## 如何开始

Herobrine 会主世界随机生成，尤其喜欢亮度≤7的黑暗环境，常在玩家附近8-16格出现。也可在聊天框输入 hb 指令立即召唤。与他的交互主要通过右键打开界面进行交易、查看信任、接取任务；也可以手持特定物品（创世神的契约）右键他来执行“传送到指定位置”、“休息”、“守卫”等指令。要提升信任，可以进行交易，完成他通过界面发布的委托或者收集传说碎片。

注意一个存档只能有一个herobrine

Herobrine 拥有动态的心态系统，会根据你的行为在多种心态间切换——观察者、守护者、审判者、恶作剧者、维护者、怀旧者、怪物之王……每种心态都深刻影响他的言语和行为。当你热爱建造、耕作、主动邀请他时，他会逐渐成为温和的守护者；而当你主动攻击他的话，他会变成审判者；如果产生许多掉落物时或者发生爆炸时，会变为维护者；如果一次性使用许多传说碎片，他会变为怀旧者；如果杀怪杀多，他会变为怪物之王。

观察者状态不会做任何额外的事情，只是观察你；守护者状态会给玩家上buff生命恢复与抗性提升；审判者状态右键他，他不会理你；恶作剧者会提升恶作剧的概率；维护者他会把过多掉落物全部清除，并且恢复因为爆炸消失的方块；怀旧者状态会在原地待着感叹；怪物之王会为怪物上buff。所有的状态会随玩家的行为动态调整 。

他的待机动画充满生命感：有时他会悬浮冥想，周身环绕神秘粒子；有时他会横握镰刀，轻轻抚摸锋刃，仿佛在追忆什么；偶尔他会抬起右手在半空中虚点，像在调试世界的底层规则；甚至会在瞬间闪烁瞬移，让你感受到他作为“代码之神”的本能。同时，他能感知环境——知道你正盯着哪块方块，知道你手里握着什么物品，并据此做出微妙反应。
在交互页面中还可以切换herobrine的皮肤。

## 丰富的互动

与 Herobrine 的关系建立在信任之上。你可以通过交易稀有材料、与他并肩作战、完成他发布的委托来逐步提升信任等级。每一级信任都会解锁新的对话和奖励：从最初的传说手册，到10级时的不死图腾，20级的永久夜视能力“幽邃之视”，50级获得奖励灵魂之契（死亡不掉落物品与经验）等等，并可以开启陪伴模式。

在陪伴模式下，Herobrine 会跟随你：根据地形自动切换浮空或步行形态，距离过远时瞬间传送回你身边，并用独特的语言与你互动。随着信任加深，还可以解锁新功能，50级时清除附近的杂草与树木（保留原始地形），70级时抹平周围地形（保留原有方块特色）。当你前往终末之环后，还可解锁和平模式，让所有怪物无视你，但一旦攻击任何生物，誓约破碎，你将面对狂暴的怪物群。即便在平时，他的君王光环也会让周围怪物停止攻击，甚至让末影龙降落在地上臣服。

你可以通过邀请机制与他互动：手持“创世神的契约”对着床、船、矿车或任意模组座椅 Shift+右键，Herobrine 便会走过来坐下，停留无限长的时间；再次对着他坐的物品 Shift+右键，他就会起身。

## 智能对话系统
Herobrine 能与玩家进行真正意义上的对话。如果你配置了云端服务（支持 deepseek等， 兼容 API），他将接入自然大语言模型，实现自由的交谈，并记住你们之前的聊天内容。若没有网络，内置的本地对话库也会通过关键词匹配回应你的话，数千条预设对话覆盖了各种情境。他还会根据你的活动主动点评：挖矿时、战斗时、生命垂危时，都有可能会与你交流。

## 世界异象与探索 

在主世界探险时，你可能偶遇不稳定地带——地形被地狱岩、灵魂土等异域方块取代，幽灵生物在此出没。这些幽灵僵尸、幽灵苦力怕、幽灵骷髅被击杀后会掉落特殊物品，可用于与 Herobrine 交易。
使用永恒门匙（可在远古城市、末地城、要塞图书馆找到）右键末地基岩，便可开启通往专属维度终末之环的传送门。这里由三层巨大的基岩圆环构成，是 Herobrine 的领域。首次进入会触发成就，右键 Herobrine 还能触发特殊对话并获得一本边界手记。
此外，世界上还游荡着两个特殊存在：幽灵史蒂夫以0.5%的极低概率生成于主世界地表，击杀后掉落“源代码碎片”；故障村民“N.” 会以64绿宝石出售屏障方块，当你退出交易页面后，他会触发惊悚音效并留下一块传说碎片，随后消失。


## 重要物品

终末之诗是 Herobrine 的镰刀，兼具斧、锹、镐、锄功能，可通过 Shift+右键切换三种形态，所有伤害都与你的信任度挂钩：

破境形态：右键发射雷枪，造成范围伤害（是否破坏地形可在配置中调整）。

鸣雷形态：右键召唤数道雷电，自动索敌攻击敌对生物。

破空形态：攻速极快，攻击时产生虚空裂缝，造成持续伤害。

其他物品包括：灵魂之契（死亡不掉落）、凌越之允（生存模式获得创造飞行）、幽邃之视（永久夜视）、回溯之石（传送至上一次死亡位置）、记忆碎片（放入唱片机召唤故障回响，引导寻找拥有永恒门匙的稀有结构），以及创世神的契约——召唤 Herobrine 的核心物品，签署后右键即可召唤。

## 背景故事搜集

模组设置了一套完整的叙事系统。通过传说手册（信任2级可领取）和散落在世界各处的11块知识碎片，你可以逐步拼凑 Herobrine 的起源。碎片获取方式各异：有的在信任奖励中直接领取，有的藏在要塞图书馆、废弃矿井，有的通过击杀原版怪物或幽灵生物获得，有的在村庄箱子、末地城、远古城市中找到，甚至还有通过睡觉触发特殊事件、一动不动站立几分钟等神秘方式解锁。当你将碎片与手册合并，不仅能阅读背景故事，Herobrine 也会“学习”这些知识，从而改变对你的态度。


## 模组兼容
Herobrine Companion 深度集成了 JEI 与 KubeJS。在 JEI 中你可以直接查看 Herobrine 的所有交易项和各信任等级的奖励预览；
通过 KubeJS 脚本，整合包作者可以动态修改奖励与交易，实现高度自定义。
KubeJS示例：


    HerobrineCompanionEvents.registerTrades(event => {
    console.info('HerobrineCompanion: Registering custom trades via KubeJS...')
    // 移除所有默认交易 (可选)
    // event.removeAll()

    // 添加简单交易: 5个苹果换1个钻石
    event.add('5x minecraft:apple', 'minecraft:diamond')

    // 添加双输入交易: 5个苹果 + 1个金锭 换 1个钻石
    event.add('5x minecraft:apple', 'minecraft:gold_ingot', 'minecraft:diamond')

    // 根据信任等级添加交易

    // 或者直接调用方法 event.getTrustLevel()
    if (event.getTrustLevel() >= 50) {
        event.add('minecraft:dirt', 'minecraft:nether_star')
        }
     })

    HerobrineCompanionEvents.registerRewards(event => {
    // 1. 修改现有奖励 (ID 2 是钻石奖励)
    event.modify(2, reward => {
        // 修改需要的信任值
        reward.requiredTrust = 5

        // 添加新物品
        reward.add('5x minecraft:emerald')
        reward.add(Item.of('minecraft:enchanted_book').enchant('sharpness', 5))

        // 注意：不要直接用 reward.items.add()，请用 reward.add()
    })

    // 2. 移除原来的图腾奖励
    event.remove(0)

    // 3. 添加一个全新的奖励
    event.add(100, 10, 'minecraft:nether_star', '3x minecraft:diamond_block')
    })

知识碎片：共11种，散落世界各地，与手册合并可解锁 Herobrine 的过往，同时他会"学习"这些知识，改变对你的态度。

战利品分布与获取指南

🗝️ 永恒门匙 (Eternal Key) —— 维度通行证
永恒门匙是前往“终末之环”维度的唯一凭证。在末地右键一个基岩就可开启通往终末之环的传送门。

获取地点：
远古城市 (Ancient City)：深暗之域的箱子中。
末地城宝藏 (End City Treasure)：末地船或高塔顶部的箱子中。
要塞图书馆 (Stronghold Library)：存放着古老知识的图书馆箱子中。
由于生成概率极高，玩家无需搜遍每一个遗迹，通常在探索完第一个完整的上述结构后即可获得该钥匙。

📜 知识碎片分布图谱 (Lore Fragments Map)
为了拼凑出 Herobrine 的完整起源，玩家需要收集散落在世界各地的知识碎片。不同的碎片对应着世界观中不同层面的“真相”。
herobrine的奖励中当信任度等级达到了2级，可领取传说手册，右键传说碎片可收集至传说手册。
